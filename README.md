Check your rate with React!
------

    $ cd server
    $ bundle install
    $ ruby react-test-server.rb
    $ cd ..
    $ python -m SimpleHTTPServer 8000

Go to [http://localhost:8000/template.html](http://localhost:8000/template.html)

If you want to dev:

    $ npm install -g react-tools
